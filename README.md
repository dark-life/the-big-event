# the-big-event

back-end management system developed using Node.js + express.js + mysql 

## introduction

根据黑马大事件项目，使用Node.js + express.js + mysql 进行后端api开发
前端部分位于web文件夹
后端部分位于api_server文件夹
数据库文件位于db.sql

## install

~~~
npm install
~~~

## tips

##### 修复前端bug

1.新建文章后左侧导航栏未跳转至文章列表

##### 修复后端bug：

1.在对应文章分类下还有文章时，可以删除文章分类
